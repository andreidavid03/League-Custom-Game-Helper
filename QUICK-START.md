# 🎮 LoL Custom Game Helper - Quick Start Guide

## For Your Friends: How to Run the App

### 📋 Prerequisites
- **Node.js** must be installed on your computer
- Download from: https://nodejs.org/ (choose LTS version)
- This will also install npm automatically

### 🚀 Quick Start (Choose one method)

#### **Windows Users:**
1. **Double-click** `start-app.bat` 
2. **OR** Right-click `start-app.ps1` → "Run with PowerShell"

#### **Mac/Linux Users:**
1. **Double-click** `start-app.sh`
2. **OR** Open Terminal and run: `./start-app.sh`

### ⏱️ What Happens Next:
1. The script will automatically install dependencies (first time only)
2. Two terminal windows will open:
   - Backend server (port 3001)
   - Frontend server (port 3000)
3. Your browser will automatically open to http://localhost:3000
4. Wait for both terminals to show "Ready" before using the app

### 🎯 Using the App:
1. **Players Tab**: Add your friends' names and preferred roles
2. **Random Teams**: Generate balanced random teams
3. **Captain Draft**: Let captains pick teams
4. **Team Selector**: Use the wheel or CS:GO-style selector
5. **Match History**: View past games and statistics

### 🛑 To Stop the App:
- Close both terminal windows that opened
- Or press `Ctrl+C` in each terminal

### 🔧 Troubleshooting:
- **Port already in use**: Close any existing instances and restart
- **Dependencies fail**: Make sure Node.js is properly installed
- **Permission denied**: On Mac/Linux, run `chmod +x start-app.sh` first
